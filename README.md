💅 Сайт-визитка для мастера маникюра Дарьи Анцифировой
🎯 О проекте:

Данный сайт разработан по индивидуальному заказу для мастера маникюра 
Анцифировой Дарьи. Проект выполнен в рамках дипломной работы и 
одновременно является реальным коммерческим продуктом.

Заказчик: Анцифирова Дарья (мастер маникюра, г. Красноярск)
Назначение: привлечение клиентов, демонстрация работ, онлайн-запись
Статус: ✓ Проект показан заказчику, но нужна доработка бд

------------------------------------------------------------------------------------------------------------------------

📋Требования заказчика:

При разработке учитывались пожелания заказчика:

-Нежная, женственная цветовая гамма (розовый, золотой)
-Обязательный раздел с фотографиями работ
-Удобная запись для клиентов любого возраста
-Адаптация под мобильные устройства (90% клиентов заходят с телефонов)
-Возможность самостоятельно менять фото в портфолио (через загрузку в папки)

Описание проекта:

Daria Nails - это современный, стильный и полностью адаптивный сайт-визитка для мастера маникюра. Проект разработан в рамках дипломной работы и представляет собой готовое решение для частного специалиста в бьюти-сфере.

Почему этот проект важен?

📱 Мобильные клиенты - сайт идеально выглядит на любых устройствах
⚡ Быстрая запись - клиент записывается в 3 клика
🎨 Визуальный контент - портфолио и прайс-лист всегда под рукой
🔒 Контроль данных - мастер сам управляет расписанием

🛠 Функционал

Для клиентов:

|     Функция:   |  Описание:  |
| 🏠 Главная | Приветственный экран с призывом к действию |
| 👩‍🎨 мастере | Информация об опыте, статистика, преимущества |
| 💅 Услуги | Полный каталог с ценами, временем и кнопкой записи |
| 📸 Портфолио | Галерея работ с модальным окном для просмотра |
| 💰 Прайс-лист | Изображения с ценами (с возможностью увеличения) |
| 📅 Онлайн-запись | Интерактивная форма с выбором услуги, даты и времени |
| 📞 Контакты | Адрес, телефон, соцсети, карта |
| 📱 Адаптивность | Идеально на всех устройствах |

Для мастера:

- ✅ Простое обновление услуг (массив данных в JS)
- ✅ Управление заблокированными слотами (объект " bookedSlots ")
- ✅ Легкая замена фото в портфолио

🛠 Технологии

Frontend:
HTML5 - семантическая вёрстка

CSS3:
- Flexbox и Grid для построения сеток
- CSS-переменные (":root") для цветовой схемы
- Медиазапросы для адаптивности
- Анимации и hover - эффекты

JavaScript (ES6+):

- Манипуляции с DOM
- Обработка событий
- Работа с массивами и объектами
- Модальные окна

Инструменты:
Google Fonts (шрифты Montserrat, Open Sans)
Font Awesome (иконки)
Яндекс.Карты / Google Maps
