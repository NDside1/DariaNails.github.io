Инструкция для мастера Дарьи
Как обновлять сайт


1) Добавить новое фото в портфолио

1. Подготовьте фото (желательно одного размера)
2. Сохраните в папку `images/portfolio/`
3. Назовите файл следующим номером (например, 10.jpg)
4. Готово! Фото появится в галерее автоматически

2) Изменить цены в услугах

Цены хранятся в JavaScript - массиве " services " в файле " index.html ".
Найдите этот массив (строка примерно 950) и измените нужные значения.

3. Заблокировать время

В объекте "bookedSlots" в коде (строка \~980) добавьте дату и время:
"""javascript
bookedSlots = {
'2026-01-15': \['10:00', '14:00'], // уже занято
'2026-03-25': \['12:30'] // добавьте свою дату
}
